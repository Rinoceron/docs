# Rhino API Reference

This reference exposes only public endpoints under `/public/*`.

## Principles

- Internal routes are excluded from this site.
- Stage pages map directly to Rhino operating stages.
- OpenAPI source is maintained in `api-reference/rhino/openapi.public.json`.

## Create party

`POST /public/parties` creates the customer `party` record after onboarding approval.

### Request contract

```json
{
  "type": "person | business",
  "legal_name": "string",
  "phone": "string",
  "email": "string"
}
```

### Success response

```json
{
  "party_id": "uuid"
}
```
